# Font Setup Required

The "Anti Design Endeavour" font needs to be downloaded manually:

1. Visit: https://befonts.com/anti-design-endeavour-font.html
2. Download the font files
3. Convert to .woff2 format if needed (use https://cloudconvert.com/ttf-to-woff2)
4. Place the file as: `public/fonts/anti-design-endeavour.woff2`

Until the font is installed, the hero "MEENAKSHI" text will fall back to the serif font stack.
